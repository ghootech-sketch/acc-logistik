# ACC Logistic Website

Company profile multipage website untuk **ACC Logistic (Antar Cargo Cepat)**.

## Stack

- Next.js 14 App Router
- Tailwind CSS v3
- TypeScript
- Google Fonts via `next/font/google`

## Pages

- `/` Beranda
- `/layanan` Layanan Kami
- `/rute` Rute Pengiriman
- `/ulasan` Ulasan Pelanggan
- `/kontak` Kontak dan Google Maps

## Development

```bash
npm install
npm run dev
```

Buka `http://localhost:3000`.

## Assets

OG image diarahkan ke `/public/assets/banner-acc.png`. Upload manual file banner ke path tersebut jika belum tersedia.
